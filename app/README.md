# Online Shopping Backend API

Backend implementation of a basic e-commerce platform using FastAPI.

## Features
- Product management
- RESTful API design
- Modular router structure
- Clean code organization

## Tech Stack
- Python
- FastAPI
- Uvicorn

## How to Run

pip install -r requirements.txt  
uvicorn app.main:app --reload

Then open:
http://127.0.0.1:8000/docs
